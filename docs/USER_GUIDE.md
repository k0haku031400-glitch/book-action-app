# BookAction ユーザーガイド

## BookAction とは

BookAction は、読んだ本の知識を「実践」に変えるためのWebアプリケーションです。AIがあなたの目標に合わせた4週間の実践プランを自動生成し、習慣化を支援します。

## はじめ方

### 1. アカウント作成

1. トップページの「無料で始める」をクリック
2. 名前、メールアドレス、パスワード（8文字以上）を入力
3. 「アカウント作成」をクリック

### 2. 本を登録

1. ダッシュボードまたは「本を登録」メニューから登録画面へ
2. 以下の情報を入力:
   - **本のタイトル**: 読んだ本の名前
   - **カテゴリー**: ビジネス、自己啓発、技術書など
   - **本の概要**: 主要内容や著者の主張
   - **キーポイント**: 印象に残った要点
   - **実践目標**: この本から何を実践したいか
3. 「登録してプランを生成」をクリック

### 3. 実践プランの実行

1. 生成されたプランの「タスク一覧」を確認
2. 第1週から順にタスクに取り組む
3. 完了したタスクにチェックを入れる
4. 振り返りを記録（任意だが推奨）

## ダッシュボードの見方

| 項目 | 説明 |
|------|------|
| レベル | 獲得ポイントに応じて上がるレベル |
| ストリーク | 連続でタスクを完了した日数 |
| 完了タスク | 累計で完了したタスク数 |
| 今日のタスク | 本日が期限のタスク一覧 |
| 全体の進捗 | すべての本の完了率 |

## ポイントシステム

タスクを完了するとポイントが獲得できます:

- 毎日タスク: **15ポイント**
- 週次タスク: **25ポイント**
- 一度きりタスク: **20ポイント**

100ポイントごとにレベルが1上がります。

## 4週間プランの構成

| 週 | 内容 |
|----|------|
| 第1週 | 基礎理解 — 本の概念を日常に取り入れる |
| 第2-3週 | 実践 — 具体的な行動タスクを実行 |
| 第4週 | 定着 — 振り返りと習慣の定着 |

## よくある質問

**Q: AIプランが生成されない**
A: 管理者が OpenAI API キーを設定している必要があります。プラン未生成の場合、本の詳細ページから手動で生成できます。

**Q: ダークモードに切り替えたい**
A: ヘッダー右上の太陽/月アイコンをクリックしてください。

**Q: 本を削除したい**
A: 本の詳細ページ右上の「削除」ボタンから削除できます。
